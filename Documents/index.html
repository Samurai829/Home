WEB Portfolio

Es una pagina el cual le he dedicado tiempo donde al dia de hoy tiene una apariencia simple y elegante para las personas que accedan para ver la informacion.

Esta web tomo un tiempo aproximado de 2 meses de constantes actualizaciones y desarrollo para lograr tener la aperiencia que tiene hoy en dia, esta desarrollada con lenguajes como HTML, CSS y JavaScript.

En el repositorio de Github esta la version 1.0 a la disponibilidad para poderse usar totalmente libre.

AUTOMATION

Es un proyecto de herramientas del Entorno DevOps y SRE, este proyecto tiene como funcion agregar la automatizacion de cada herramienta solo usando un simple comando para ejecutarlas.

El proyecto es algo personal que me di la tarea de hacer ya que esto reducce el margen de error de instalacion y ejecucion.

Este proyecto se encuentra en repositorio de Github.

CONTAINERS

Es un proyecto de CONTENEDORES DOCKER con servicios del Entorno DevOps y SRE, este proyecto tiene como principal tarea simplificar la instalacion de cada herramienta para la ejecucion y uso mediante DOCKER.

Al igual que la mayoria de mis proyectos es algo personal que me di la tarea de hacer ya que esto reducce el margen de error de instalacion y ejecucion.

Este proyecto se encuentra en repositorio de Github.
